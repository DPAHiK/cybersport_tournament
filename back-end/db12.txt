User:
POST: /user/ - добавление пользователя (администратор)
GET: /user/{id} - просмотр информации о пользователе (авторизованный пользователь)
PUT: /user/{id} - изменить информацию о пользователе (администратор)
DELETE: /user/{id} - удалить пользователя (администратор)
GET: /user/ - список всех пользователей (администратор)
POST: /signup/ - регистрация (любой пользователь)
POST: /login/ - вход (любой пользователь)
POST: /logout/ - выход (авторизованный пользователь)

Team:
GET: /team/ - список всех команд (неавторизованный пользователь)
GET: /team/{id} - информация о команде (авторизованный пользователь)
PUT: /team/{id} - изменить информацию о команде (пользователь-игрок(если в команде))
POST: /team/ - создать команду (пользователь-игрок)
DELETE: /team/{id} - удалить команду (пользователь-игрок(если в команде))

TeamMember:
GET: /team/{teamId}/member/ - список всех членов команды (авторизованный пользователь)
POST: /team/{teamId}/member/ - добавление игрока в команду (администратор)
DELETE: /team/{teamId}/member/{teamMemberId}/ - удаление игрока из команды (администратор)

Tournament:
GET: /tournament/ - список всех турниров (неавторизованный пользователь)
GET: /tournament/{id} - информация о турнире (авторизованный пользователь)
POST: /tournament/ - добавление турнира (пользователь-организатор)
PUT: /tournament/{id} - изменить информацию о турнире(пользователь-организатор(если создал))
DELETE: /tournament/{id} - удаление информации о турнире(пользователь-организатор(если создал))

EngagedTeam:
GET: /tournament/{tournamentId}/team - список команд-участников турнира (авторизованный пользователь)
GET: /team/{teamId}/tournament - список турниров, в которых команда принимала участие (авторизованный пользователь)
POST: /tournament/{tournamentId}/team - добавление команды-участника в турнир (администратор)
DELETE: /tournament/{tournamentId}/team/{teamID} - удаление команды-участника из турнира (администратор)

TeamQuery:
GET: /team/query - список всех заявок на участие (авторизованный пользователь)
GET: /team/{teamId}/query - список всех заявок команды (авторизованный пользователь)
GET: /team/{teamId}/query/{queryId} - конкретная заявка команды (авторизованный пользователь)
POST: /team/{teamId}/query - подать заявку ( пользователь-игрок)
DELETE: GET: /team/{teamId}/query/{queryId} - удалить заявку команды (администратор)

TournamentResult:
GET: /tournament/result - результаты всех турниров (неавторизованный пользователь)
GET: /tournament/{tournamentId}/result - результат турнира (авторизованный пользователь)
POST: /tournament/{tournamentId}/result - добавление результата турнира (администратор)
DELETE: /tournament/{tournamentId}/result - удаление результата турнира (администратор)

Match:
GET: /tournament/{tournamentId}/match - все матчи турнира (авторизованный пользователь)
GET: /tournament/{tournamentId}/match/{matchId} - один матч турнира (авторизованный пользователь)
POST: /tournament/{tournamentId}/match - добавить матч турнира (администратор)
PUT: /tournament/{tournamentId}/match/{matchId} - изменить матч турнира (администратор)
DELETE: /tournament/{tournamentId}/match/{matchId} - удалить матч турнира (администратор)
